项目下载和运行
----

- 拉取项目代码
```bash
git clone https://github.com/zhangmrit/ruoyi-ant.git
cd ruoyi-ant
```

- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run serve
```

- 编译项目
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
